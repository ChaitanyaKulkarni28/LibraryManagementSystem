Install Wampserver 3, the default username is "root" (without quotes) and there is no password, so while installing you must leave the form Password box empty.

BEFORE installing, disable or close some applications:
- Close Skype or force not to use port 80

After installing wamp copy the application files into Wamp's www folder (C:\wamp64\www). Start wamp server and confirm whether all services are running or not.
Then open the browser and type http://localhost/Project1/index.php it will load the homepage of library application. The other way is open project directory and use open in browser option. After this you are good to use library application.
